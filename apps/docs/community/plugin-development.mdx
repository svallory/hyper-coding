## Basic Plugin Development

### Creating Simple Template Helpers

Add custom template functions with basic JavaScript:

```javascript
// helpers.js - Simple template helpers
export const helpers = {
  // String formatting
  capitalize: (str) => str.charAt(0).toUpperCase() + str.slice(1),
  
  // Case conversion
  kebabCase: (str) => str.replace(/\s+/g, '-').toLowerCase(),
  
  // Date utilities
  currentDate: () => new Date().toISOString().split('T')[0],
  
  // Simple pluralization
  pluralize: (word, count) => count === 1 ? word : word + 's'
};
```

Use in templates:

```ejs
---
to: components/<%= kebabCase(name) %>.tsx
---
// Generated on <%= currentDate() %>
export const <%= capitalize(name) %> = () => {
  return <div>Hello World</div>;
};
```

### Claude Code Integration

Connect HyperDev with Claude Code for enhanced development workflow:

```javascript
// hyperdev.config.js - Claude Code integration
export default {
  sources: [
    { name: 'local', path: './_templates' }
  ],
  
  // Git hooks integration
  hooks: {
    'pre-commit': ['hyper gen validate'],
    'post-commit': ['hyper gen cleanup']
  }
};
```

### Basic Plugin Configuration

```javascript
// hyperdev.config.js
export default {
  // Local template helpers
  helpers: './helpers.js',
  
  // Quality gates
  quality: {
    syntax_validation: true,
    security_scanning: true
  },
  
  // Simple integrations
  integrations: {
    claude_code: true
  }
};
```
