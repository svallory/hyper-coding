# Package.json Scripts Integration

Streamline your development workflow by integrating HyperDev commands into your package.json scripts.

## Basic Script Integration

Add HyperDev commands to your package.json scripts:

```json
{
  "scripts": {
    "dev": "concurrently \"hyperdev dev --watch\" \"next dev\"",
    "build": "hyperdev validate --all && next build",
    "gen": "hyperdev gen",
    "gen:component": "hyperdev gen component",
    "gen:page": "hyperdev gen page",
    "gen:api": "hyperdev gen api-route",
    "validate": "hyperdev validate --all",
    "templates:update": "hyperdev update-templates",
    "templates:list": "hyperdev list",
    "hyperdev:init": "hyperdev init --interactive",
    "hyperdev:status": "hyperdev status --detailed",
    "postinstall": "hyperdev setup --verify"
  }
}
```

## Development Workflow Scripts

Create scripts for different development phases:

```json
{
  "scripts": {
    "start:dev": "concurrently \"hyperdev dev --watch\" \"npm run dev\"",
    "start:staging": "hyperdev generate --env staging && npm run build && npm start",
    "start:prod": "hyperdev generate --env production --optimize && npm run build && npm start",
    
    "test": "hyperdev validate --all && jest",
    "test:templates": "hyperdev test-templates --all",
    "test:generated": "hyperdev validate-generated && npm run test",
    
    "lint": "hyperdev validate --all && eslint src/",
    "lint:fix": "hyperdev validate --all && eslint src/ --fix",
    "lint:templates": "hyperdev lint-templates",
    
    "typecheck": "hyperdev generate --types-only && tsc --noEmit",
    "typecheck:watch": "hyperdev dev --types-watch & tsc --noEmit --watch"
  }
}
```

## Advanced Configuration with HyperDev

Include HyperDev configuration directly in package.json:

```json
{
  "scripts": {
    "dev": "hyperdev dev --watch & next dev",
    "build": "hyperdev validate --all && next build"
  },
  "hyperdev": {
    "templates": {
      "component": "@hyperdev/react-component",
      "page": "@hyperdev/next-page",
      "api": "@hyperdev/api-route"
    },
    "qualityGates": {
      "enabled": true,
      "onBuild": true,
      "rules": ["typescript", "eslint", "security"]
    },
    "environments": {
      "development": {
        "features": {
          "debugging": true,
          "hotReload": true
        }
      },
      "production": {
        "features": {
          "optimization": true,
          "minification": true
        }
      }
    }
  }
}
```
