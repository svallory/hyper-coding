# hyper list

Browse and list available resources across the HyperDev ecosystem with powerful filtering, sorting, and categorization options for templates, epics, components, and project assets.

## Syntax

```bash
hyper list [resource] [options]
```

## Overview

The `hyper list` command provides comprehensive resource browsing:

- **Template catalog**: Browse available templates by category and technology
- **Epic management**: List and manage project epics and their status
- **Project inventory**: View components, modules, and assets in your project
- **Resource discovery**: Explore available tools, patterns, and solutions
- **Advanced filtering**: Filter by technology, trust level, category, and status
- **Detailed information**: View metadata, compatibility, and usage statistics

<Info>
  Use `hyper list` to discover what's available before searching or generating. It provides
  organized, categorical views of resources with detailed filtering options.
</Info>

## Options

| Option | Description | Default | Example |
|--------|-------------|---------|---------|
| `--detailed, -d` | Show detailed information | `false` | `--detailed` |
| `--trust-scores` | Include trust scores | `true` | `--trust-scores` |
| `--official-only` | Only official resources | `false` | `--official-only` |
| `--filter <criteria>` | Filter results | - | `--filter react` |
| `--sort <field>` | Sort by field | `name` | `--sort popularity` |
| `--category <cat>` | Category filter | `all` | `--category frontend` |
| `--json` | JSON format output | `false` | `--json` |
| `--limit <n>` | Limit results | `50` | `--limit 20` |

## Examples

### Basic Listing


```bash
# List React templates only
hyper list templates --filter react

# List high-trust templates
hyper list templates --trust-scores --filter "trust>8"

# List official resources only
hyper list --official-only

# List by category
hyper list templates --category frontend
```

### Detailed Information

```bash
# Show detailed template information
hyper list templates --detailed

# Include trust scores and metrics
hyper list templates --detailed --trust-scores

# Sort by popularity
hyper list templates --sort popularity --detailed
```

## Advanced Filtering

### Filter Syntax

```bash
# Simple text filter
hyper list templates --filter "react"

# Multiple criteria
hyper list templates --filter "react,typescript"

# Comparison operators
hyper list templates --filter "trust>8"
hyper list templates --filter "downloads>1000"
hyper list templates --filter "updated<30d"

# Complex filters
hyper list templates --filter "trust>8 AND category=frontend"
```

**Available Filter Fields**:
- `trust`: Trust score (1-10)
- `downloads`: Download count
- `stars`: GitHub stars
- `updated`: Last update (use time suffixes: d, w, m)
- `category`: Template category
- `technology`: Technology stack
- `author`: Author or organization
- `official`: Official status (true/false)

### Sorting Options

```bash
# Sort by different criteria
hyper list templates --sort name          # Alphabetical
hyper list templates --sort trust         # Trust score (desc)
hyper list templates --sort popularity    # Downloads/stars
hyper list templates --sort updated       # Most recent first
hyper list templates --sort relevance     # Context relevance
```

### Category Filtering

```bash
# Frontend-specific templates
hyper list templates --category frontend

# Backend-focused resources
hyper list templates --category backend

# DevOps and infrastructure
hyper list templates --category devops

# Full-stack solutions
hyper list templates --category fullstack
```

## Integration Features

### Quick Actions

Each listing includes quick action suggestions:

```bash
# From any list result, you can:
hyper generate <template-name>     # Generate from template
hyper search <template-name>       # Search for similar

```

### Bookmarking

```bash
# Bookmark frequently used resources
hyper list templates --bookmark react-component

# List bookmarked resources
hyper list --bookmarked

# Remove bookmarks
hyper list --bookmarks --remove react-component
```

### Comparison

```bash

# Show differences
hyper list templates --diff template-a template-b
```

## Configuration

### List Display Settings

```javascript
// hyperdev.config.js
export default {
  list: {
    // Default display options
    default_view: 'summary',        // summary, detailed, compact
    show_trust_scores: true,
    show_popularity: true,
    show_last_updated: true,
    
    // Default limits
    default_limit: 50,
    max_limit: 200,
    
    // Sorting preferences
    default_sort: 'relevance',      // name, trust, popularity, updated
    
    // Filtering preferences
    default_trust_level: 6,
    include_experimental: false,
    
    // Categories to show
    categories: [
      'frontend',
      'backend', 
      'fullstack',
      'devops',
      'mobile'
    ]
  }
};
```
