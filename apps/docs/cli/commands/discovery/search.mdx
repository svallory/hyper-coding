# hyper search

Powerful universal search command that helps you find kits, cookbooks, and templates across the entire HyperDev ecosystem with intelligent filtering and ranking.

## Syntax

```bash
hyper search <query> [options]
```

## Overview

The `hyper search` command provides comprehensive search capabilities:

- **Template discovery**: Find the perfect template for your needs
- **Pattern matching**: Discover architectural patterns and best practices
- **Solution finding**: Search for solutions to common development problems
- **Documentation search**: Find relevant guides, tutorials, and references
- **Intelligent ranking**: Results ranked by relevance, trust score, and popularity
- **Context awareness**: Search results tailored to your project and tech stack

<Info>
  Search results include trust scores, popularity metrics, and compatibility information to help you
  make informed decisions about which resources to use.
</Info>

## Options

| Option | Description | Default | Example |
|--------|-------------|---------|---------|
| `--type <type>` | Search type filter | `all` | `--type templates` |
| `--scope <scope>` | Search scope | `all` | `--scope solutions` |
| `--language <lang>` | Programming language filter | - | `--language typescript` |
| `--framework <name>` | Framework filter | - | `--framework react` |
| `--trust-level <min>` | Minimum trust level (1-10) | `6` | `--trust-level 8` |
| `--official-only` | Only official resources | `false` | `--official-only` |
| `--limit <n>` | Limit results | `20` | `--limit 10` |
| `--json` | JSON format output | `false` | `--json` |

## Examples

### Basic Search

```bash
# Search for React components
hyper search "react component"

hyper search "user authentication"

hyper search "rest api typescript"
```

```bash
# Templates only

hyper search "payment processing" --trust-level 9

hyper search "database migration" --official-only
```

```bash
# TypeScript-focused results
hyper search "form validation" --language typescript

# React ecosystem only
hyper search "state management" --framework react

hyper search "api authentication" --language javascript --scope backend
```

## Advanced Search Features

### Context-Aware Search

HyperDev automatically considers your project context:

```bash
# In a Next.js project
hyper search "authentication"
# Results prioritize Next.js-compatible auth solutions

# In a React Native project  
hyper search "navigation"
# Results focus on mobile navigation patterns
```

**Context Factors**:
- **Current tech stack**: Detected from package.json and project files
- **Project type**: Web app, mobile app, API service, library, etc.
- **Team preferences**: Previously used templates and patterns
- **Active epic**: Current epic context influences results

### Fuzzy and Semantic Search

```bash
# Fuzzy matching handles typos
hyper search "recat compnent"  # Finds "react component"

# Semantic understanding
hyper search "user login system"  # Finds auth, authentication, login templates

# Natural language queries
hyper search "I need a way to handle payments in my app"
```

### Search with Composition

```bash
# Find composable solutions
hyper search "frontend backend database" --composable

# Multi-technology search
hyper search "react express postgresql" --stack

# Full-stack search
hyper search "saas application" --complete
```

## Search Filters and Refinement

### Trust Level Filtering

```bash
# Only highly trusted resources
hyper search "authentication" --trust-level 9

# Community resources with moderate trust
hyper search "ui components" --trust-level 7

# Include experimental resources
hyper search "bleeding edge react" --trust-level 5
```

**Trust Level Guide**:
- **10**: Official HyperDev resources
- **9**: Verified community leaders and companies  
- **8**: Well-tested community resources
- **7**: Popular community resources
- **6**: Standard community resources
- **5 and below**: Experimental or newer resources

### Technology Filtering

```bash
# Language-specific search
hyper search "web scraping" --language python
hyper search "api client" --language typescript

# Framework-specific search  
hyper search "form handling" --framework react
hyper search "routing" --framework vue

# Multi-technology filtering
hyper search "fullstack" --language typescript --framework next
```

### Scope Filtering

```bash
# Frontend-only results
hyper search "component library" --scope frontend

# Backend-focused results
hyper search "authentication" --scope backend

# DevOps and infrastructure
hyper search "deployment" --scope devops

# Full-stack solutions
hyper search "saas starter" --scope fullstack
```

## Search Result Details

### Understanding Search Results

Each search result includes:

- **Trust Score**: Reliability and quality rating (1-10)
- **Popularity Indicators**: Download count, stars, usage metrics
- **Freshness**: Last update, maintenance status
- **Compatibility**: Technology compatibility and requirements
- **Author Information**: Creator, maintainer, organization
- **Quick Actions**: Direct commands to use the resource

### Result Ranking Algorithm

Results are ranked by:

1. **Relevance Score**: Query match quality and semantic similarity
2. **Trust Level**: Resource reliability and quality assessment
3. **Popularity**: Community adoption and usage metrics
4. **Freshness**: Recent updates and active maintenance
5. **Context Fit**: Compatibility with your project and preferences
6. **User Behavior**: Previous successful usage by similar projects

## Integration with Other Commands

### Search-to-Generate Workflow

```bash
# Search and directly generate
hyper search "react form" --limit 3
# Choose result and generate:
hyper generate react-hook-form-template

# Interactive search and generate
hyper generate --search "payment form"
```

### Search-to-Plan Workflow

```bash
# Search for epic templates
hyper search "user management epic" --type epic-templates

# Search for architecture patterns during planning
hyper plan user-system --search "user authentication patterns"
```

### Saved Searches

```bash
# Save frequently used searches
hyper search "react components" --save "my-react-search"

# Use saved searches
hyper search --load "my-react-search"

# List saved searches
hyper search --saved --list
```

## Configuration

### Search Settings

```javascript
// hyperdev.config.js
export default {
  search: {
    // Default search behavior
    default_trust_level: 7,
    max_results: 20,
    include_experimental: false,
    
    // Search sources
    sources: [
      'official',        // Official HyperDev resources
      'community',       // Community templates and patterns
      'github',          // GitHub repositories
      'npm',            // NPM packages
      'documentation'   // Documentation and guides
    ],
    
    // Personalization
    personalization: {
      enabled: true,
      learn_preferences: true,
      boost_used_technologies: true,
      boost_team_resources: true
    },
    
    // Output preferences
    output: {
      show_trust_scores: true,
      show_popularity: true,
      show_quick_actions: true,
      color_output: true
    }
  }
};
```
