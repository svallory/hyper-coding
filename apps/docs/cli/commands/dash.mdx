# hyper dash

Launch a simple monitoring dashboard for basic project overview.

## Syntax

```bash
hyper dash [options]
```

## Overview

The `hyper dash` command provides basic project monitoring:

- **Simple monitoring**: Basic project status and progress
- **Plan progress**: View plan completion status
- **Development status**: Monitor development progress

<Info>
  The dashboard provides a simple overview of your HyperDev project status.
</Info>

## Basic Views

| View | Description |
|------|-------------|
| `overview` | Project overview and status |
| `plans` | Plan progress tracking |

## Options

| Option | Description | Default |
|--------|-------------|---------|
| `--refresh <seconds>` | Auto-refresh interval | `30` |

## Examples

### Basic Usage

```bash
# Launch dashboard
hyper dash

# Launch with custom refresh rate  
hyper dash --refresh 10
```
