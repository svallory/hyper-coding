## mintlify-tsdocs package

Mintlify-TSdocs generates Mintlify-compatible MDX documentation from .api.json files created by API Extractor. The `mintlify-tsdocs` package provides the command-line tool for generating documentation with proper frontmatter and navigation integration for Mintlify documentation sites.
