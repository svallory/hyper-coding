## How to Use These Examples

### 1. Find Your Use Case
Browse the categories above or use the search to find examples that match your needs.

### 2. Copy and Adapt
Each example includes:
- Complete working code
- Expected output structure
- Customization instructions
- Explanation of key concepts

### 3. Test Locally
```bash
# Copy the template files to your project
# Run hyper gen to test the generation
hyper gen example-name --dry-run
```

### 4. Customize for Your Needs
- Modify variables and prompts
- Add your own template logic
- Extend with additional files
- Integrate with your development workflow
