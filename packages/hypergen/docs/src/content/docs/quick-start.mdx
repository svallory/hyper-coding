# Quick Start

Get Hypergen installed and generate your first files in under 5 minutes.

## Installation

```bash
bun add hypergen
```

Or with npm:

```bash
npm install hypergen
```

## Your First Recipe

Execute a recipe to generate files:

```bash
hypergen recipe execute my-component.yml --name=Button --typescript=true
```

## Explore Commands

### Recipe Commands

```bash
# Validate a recipe
hypergen recipe validate my-component.yml

# Get recipe information
hypergen recipe info my-component.yml

# List recipe steps
hypergen step list my-component.yml
```

### Discovery Commands

```bash
# List available actions
hypergen action list

# Discover cookbooks
hypergen discover react
```
