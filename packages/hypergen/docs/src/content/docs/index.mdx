## A modern code generator for the age of AI

Hypergen orchestrates templates, actions, and codemods through recipes. Built with TypeScript, featuring LiquidJS templating, decorator-based actions, and intelligent cookbook discovery.

## Key Features

- **Pluggable Engines** - Use LiquidJS, EJS, or plug your own engine
- **Recipe Step System** - Orchestrate Templates, Actions, CodeMods, and Recipes in sequential steps with conditionals
- **LiquidJS & EJS Templates** - Fast, secure engines with rich filters (camelCase, snakeCase, pluralize)
- **TypeScript Actions** - Decorator-based actions with auto-discovery and type safety
- **Multi-Source Discovery** - Find cookbooks via NPM (`@hyper-kits/*`), GitHub topics, or direct URLs
- **Security First** - Creator trust system with transparent consent and validation
- **Performance Optimized** - 100ms startup through lazy loading and multi-level caching
