Hypergen is a modern code generator that orchestrates templates, actions, and codemods through recipes. This guide gets you up and running quickly.

## Before You Start

Make sure you have Node.js 18+ installed:

```bash
node --version  # Should show v18 or higher
```

## Installation

<Tabs>
  <Tab title="Using Bun (Recommended)">
    ```bash
    # Install with bun
    bun add -g hypergen

    # Verify installation
    hypergen --version
    ```
  </Tab>
  <Tab title="Using npm">
    ```bash
    # Install with npm
    npm install -g hypergen

    # Verify installation
    hypergen --version
    ```
  </Tab>
</Tabs>

## Your First Generation

Let's generate your first files to see Hypergen in action:

<Steps>
  <Step title="Create a test directory">
    ```bash
    mkdir hypergen-test && cd hypergen-test
    ```
  </Step>

  <Step title="Generate a React component">
    ```bash
    # Uses built-in cookbook
    hypergen react component --name=HelloWorld --typescript=true
    ```
  </Step>

  <Step title="Verify the output">
    **Success!** You should now have:
    - `src/components/HelloWorld.tsx` - Your React component
    - `src/components/HelloWorld.test.tsx` - Basic test file
  </Step>
</Steps>

## Understanding What Happened

When you ran that command, Hypergen:

<Steps>
  <Step title="Found the recipe">
    Located the `react/component` recipe from built-in cookbooks
  </Step>
  <Step title="Processed variables">
    Used `name=HelloWorld` and `typescript=true`
  </Step>
  <Step title="Executed steps">
    Ran each step in the recipe sequentially
  </Step>
  <Step title="Generated files">
    Created files from templates with your variables
  </Step>
</Steps>

## Explore Available Commands

<CodeGroup>
  ```bash List Recipes
  # List all available recipes
  hypergen recipe list
  ```

  ```bash Recipe Info
  # Get details about a specific recipe
  hypergen recipe info react/component
  ```

  ```bash Step List
  # See what steps a recipe will execute
  hypergen step list react/component
  ```
</CodeGroup>

## Configuration (Optional)

Create `hypergen.config.js` in your project root for custom settings:

```javascript hypergen.config.js
// hypergen.config.js
export default {
  // Where to look for cookbooks
  cookbooks: ['recipes', 'cookbooks', '_templates'],

  // Template engine preferences
  engines: {
    default: 'liquidjs',
    fallback: 'ejs'
  }
};
```
