## A modern code generator for the age of AI

Hypergen orchestrates templates, actions, and codemods through recipes. Built with TypeScript, featuring LiquidJS templating, decorator-based actions, and intelligent cookbook discovery.

## Key Features

<AccordionGroup>
  <Accordion title="Pluggable Engines" icon="plug">
    Use LiquidJS, EJS, or plug your own engine for template processing
  </Accordion>
  <Accordion title="Recipe Step System" icon="list">
    Orchestrate Templates, Actions, CodeMods, and Recipes in sequential steps with conditionals
  </Accordion>
  <Accordion title="LiquidJS & EJS Templates" icon="file-code">
    Fast, secure engines with rich filters (camelCase, snakeCase, pluralize)
  </Accordion>
  <Accordion title="TypeScript Actions" icon="code">
    Decorator-based actions with auto-discovery and type safety
  </Accordion>
  <Accordion title="Multi-Source Discovery" icon="magnifying-glass">
    Find cookbooks via NPM (`@hyper-kits/*`), GitHub topics, or direct URLs
  </Accordion>
  <Accordion title="Security First" icon="shield">
    Creator trust system with transparent consent and validation
  </Accordion>
  <Accordion title="Performance Optimized" icon="bolt">
    100ms startup through lazy loading and multi-level caching
  </Accordion>
</AccordionGroup>
